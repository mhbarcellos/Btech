# BTech

Site institucional da **BTech**, empresa focada no desenvolvimento de sites, landing pages, aplicativos e sistemas digitais.

O projeto foi criado com uma estética tecnológica, escura e moderna, usando React, Vite, Tailwind CSS, Framer Motion e Lucide React.

## Link do site

https://btech-io.netlify.app

## Sobre o projeto

O site apresenta os serviços da BTech, projetos reais desenvolvidos, processo de trabalho, diferenciais, informações sobre a empresa e canais de contato.

A proposta visual segue uma linha premium e tecnológica, com fundo escuro, detalhes em azul e ciano, animações suaves, cards responsivos e mockups visuais criados com componentes.

## Como rodar o projeto localmente

Clone o repositório:

```bash
git clone https://github.com/mhbarcellos/Btech.git
```

Entre na pasta do projeto:

```bash
cd Btech
```

Instale as dependências:

```bash
npm install
```

Rode o projeto em ambiente de desenvolvimento:

```bash
npm run dev
```

Abra no navegador:

```text
http://localhost:5173
```

## Como gerar a versão de produção

```bash
npm run build
```

Para visualizar a versão de produção localmente:

```bash
npm run preview
```

## Deploy

O projeto está publicado na Netlify.

Configuração usada:

```text
Build command: npm run build
Publish directory: dist
```
